Which of the following is NOT a React Hook?

Select the correct answer from the options below.

A
useState
B
useEffect

useComponent

Correct!

useComponent is not a built-in React Hook. The standard hooks include useState, useEffect, useContext, useReducer, useCallback, useMemo, useRef, useImperativeHandle, useLayoutEffect, and useDebugValue.
